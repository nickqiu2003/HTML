# 基于web服务功能的网页控制案例

这是一个适配西门子S7-1500/1200PLC的web页面控制的演示案例，通过这个案例可以利用S7-1500/1200PLC的web服务器功能，实现远程设备(计算机、平板、手机等)通过web页面完成对PLC的远程控制功能；
为了便于验证实际效果，需要选用一台实际西门子CPU1511C-1PN的硬件PLC作为演示硬件。
